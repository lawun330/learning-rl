# Reinforcement Learning Projects

This repository reflects my learning journey in reinforcement learning. 

## Coursera + University of Alberta: Reinforcement Learning Specialization

I was introduced to reinforcement learning through the `University of Alberta: Reinforcement Learning Specialization` on Coursera.
Assignments and projects from this specialization are organized under the directory `Coursera & University of Alberta`.

The specialization consists of four courses, each divided into multiple modules.
- Course 1: Fundamentals of Reinforcement Learning
  - Module 1
  - Module 2
  - ...
- Course 2: Sample-based Learning Methods
- Course 3: Prediction and Control with Function Approximation
- Course 4: A Complete Reinforcement Learning System (Capstone)

The majority of the files and materials are provided by default. However, the assignment notebooks include my own solutions to the exercises and projects.
